# Compose ERD

Author: Daniel Rudaev (D1DX)

Generates Entity Relationship Diagrams of your Airtable base structure.

## Usage

Interactive script that analyzes base schema and creates ERD visualization.

## Setup

1. Copy `src/compose-erd.js`
2. Create an Airtable extension in your base
3. Paste into extension code editor
4. Run to generate ERD

## Output

- Visual representation of table relationships
- Field types and connections
- Link field mappings

## Use Cases

- Documenting base architecture
- Understanding complex relationships
- Planning schema changes
