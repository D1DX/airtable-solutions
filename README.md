# Airtable Solutions

Production-tested Airtable automation scripts, extension scripts, formulas, and formula builders. Built from real operational workflows and digital transformation projects delivered through [D1DX](https://d1dx.com).

Each solution includes setup instructions, versioning, and source code ready to deploy in your Airtable base.

## Contents

| Category | Count | Description |
|---|---|---|
| [Automation Scripts](./automation-scripts/) | 9 | Server-side scripts for data sync, deduplication, HTTP requests, and workflow automation |
| [Extension Scripts](./extension-scripts/) | 5 | Client-side extensions for ERD generation, webhook management, and cross-table operations |
| [Formulas](./formulas/) | 2 | Field-level formulas for indexing and change tracking |
| [Formula Builders](./formula-building-scripts/) | 2 | Python scripts that programmatically generate complex Airtable formulas |

## Highlights

- **merge-duplicates** — Identify and merge duplicate records across configurable match fields
- **parent-children-sync** — Keep parent and child records in sync across linked tables
- **compose-erd** — Generate entity-relationship diagrams from your Airtable schema
- **webhooks-api** — Full lifecycle management for Airtable webhooks (create, delete, refresh, recreate)
- **multi-table-merger** — Cross-table record synchronization, available as both automation and extension

## Usage

Browse the category folders, select a solution, and follow its README for setup instructions. Each solution is self-contained with its own documentation and version tracking.
